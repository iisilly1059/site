# âœ¨ Features
This isn't just a simple page of links. It includes:

Multi-View Interface: Navigate between Home, Games, and Settings without a page refresh.

Game Launcher: Loads games directly on the page in an iframe (or in a new tab for external sites like Krunker).

Live Game Search: Instantly filter the game list by typing in the search bar.

Cool Home Page: Features an animated "typewriter" effect and an auto-rotating game showcase.

Extensive Settings Panel:

Theme Switcher: 7+ themes including Dark, Hacker, Neon, and Cotton Candy.

Particle Effects: Toggle animated background particles and change their density.

Performance Mode: A quick toggle to reduce animations for faster use.

Panic Key: Set a custom key to quickly redirect to another site (like Google Docs).

Tab Cloaking: Disguise the site's browser title and icon (favicon).

about:blank Loader: Opens the site in a blank tab to hide it from your browser history.
